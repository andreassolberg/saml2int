SAML2int

The (SAML2Int) Interoperable SAML 2.0 Profile is a SAML 2.0 WebSSO Deployment Profile.

SAML 2.0 leaves the deployer a lot of options, like how to pass attributes, what binding to use, how to use PKI, what should be signed and what should be encrypted.

An unwanted effect of all these choices is that two deployments of software supporting SAML 2.0 may not work as smooth together as expected.

This profiles defines a minimum set of bindings and rules that needs to be followed, for those entities that are requested to follow this profile.
